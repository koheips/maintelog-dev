v18 新機能
・3タブ構成（入力 / 履歴 / 設定）
  - 設定タブ：作業追加・マスター管理・区分管理・データ・アプリ名を集約
・推奨作業
  - 入力タブ最上部に移動
  - 要対応（期限超過・70%以上経過）のみ表示、全て余裕ありなら非表示
  - 次回予定日を自動計算して表示
  - トリガー種別：「経過日数」と「累計滞在日数」を作業ごとに個別設定
  - 累計滞在日数トリガー：「累計 4日 / 5日」形式で表示
・「泊数」→「滞在日数」に名称変更
・履歴：縦タイムライン形式（横スライド廃止）
  - 区分ごとの色つきpillで識別
  - 行タップで編集・削除ボタンが展開（常時表示をやめてスッキリ）
・UIフラット化
  - セグメントコントロール（iOSスタイルタブ）
  - ボーダーレス入力フィールド
  - モーダルが下からシート型で表示
  - 保存ボタンを大型化・全幅に

キャッシュクリア手順（反映されない場合）
Chrome: F12 → Application → Storage → Clear site data
iPhone: 設定 → Safari → 履歴とWebサイトデータを消去
